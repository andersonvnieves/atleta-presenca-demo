Olá!

Esta é uma demo de aplicação simples onde vamos cadastrar atletas e marcar presença nos treinos.

Tecnologias utilizadas

Front-end

O front-end foi construído de forma simples, sem utilizar frameworks complexos de Javacript já que a demo visa introduzir os conceitos mais básicos de HTML CSS e Javascript e de como a Web funciona.

Back-end

O back-end feito em Java utiliza o spring framework para lidar com requisições http mas a camada de acesso a dados utiliza JDBC direto, novamente para introduzir menos conceitos complexos visando o entendimento básico de acesso a dados.
